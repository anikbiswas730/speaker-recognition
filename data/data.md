# Text-Independent Speaker Recognition System

This repository contains the MATLAB implementation of a text-independent speaker recognition system. It was developed as a project for the EEE-312 Digital Signal Processing (DSP) laboratory course at the Bangladesh University of Engineering and Technology (BUET).

The system identifies individuals based on their unique acoustic signatures by extracting Mel-Frequency Cepstral Coefficients (MFCC) and fundamental frequency (pitch) via the YIN algorithm, and classifying them using the Mahalanobis distance metric.

## Dataset

The audio dataset used for training and testing this system consists of 2-minute voice samples from 4 individuals, recorded at 44.1 kHz.
